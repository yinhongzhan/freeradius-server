00027 /* NOTES:
00028 
00029    This code makes no attempt to be fast! In fact, it is a very
00030    slow implementation
00031 
00032    This code is NOT a complete DES implementation. It implements only
00033    the minimum necessary for SMB authentication, as used by all SMB
00034    products (including every copy of Microsoft Windows95 ever sold)
00035 
00036    In particular, it can only do a unchained forward DES pass. This
00037    means it is not possible to use this code for encryption/decryption
00038    of data, instead it is only useful as a "hash" algorithm.

00040    There is no entry point into this code that allows normal DES operation.
