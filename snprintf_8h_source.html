00055 /*
00056  * For the FLOATING POINT FORMAT :
00057  *  the challenge was finding a way to
00058  *  manipulate the Real numbers without having
00059  *  to resort to mathematical function(it
00060  *  would require to link with -lm) and not
00061  *  going down to the bit pattern(not portable)
00062  *
00063  *  so a number, a real is:
00064 
00065       real = integral + fraction
00066 
00067       integral = ... + a(2)*10^2 + a(1)*10^1 + a(0)*10^0
00068       fraction = b(1)*10^-1 + b(2)*10^-2 + ...
00069 
00070       where:
00071        0 <= a(i) => 9
00072        0 <= b(i) => 9
00073 
00074     from then it was simple math
00075  */
